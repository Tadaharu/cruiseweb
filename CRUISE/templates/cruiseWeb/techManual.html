Technologies Behind

Overall Architecture


There are five main components in the system which includes:

  • Crawler: Twitter API crawler to crawl the traffic data
  • NLP: Natural Language Processes where the data is being extracted and interpreted
  • Phone/End node: An android phone that runs the software solution
  • User: Participant of the system
  • Web Service: Interacts with the database and the phone

NLP

Named Entity Recognition


Named Entity Recognition (NLP) is the process of extracting named entity (which commonly are Nouns) from the text. The NER system in tweaked to add the extra ability in extraction of traffic state.


Contextual Analysis


Contextual Analysis is the process to understand the text in sentence level. In this project, contextual analysis is cruitial in relating the relationship of the extracted data for analysis.


Reconciliation

There is possibilities that the extracted data contradicts with each other. This is where the reconciliation takes place. This process will weight the data and make a conclusion for the traffic state for the respective data.
